BinaryPointer Type

Namespace: B2R2.FrontEnd.BinFile
Assembly: B2R2.FrontEnd.BinFile.dll
Base Type: ValueType
All Interfaces: IEquatable<BinaryPointer> , IStructuralEquatable , IComparable<BinaryPointer> , IComparable , IStructuralComparable
Kind: Struct

A pointer to binary, which is used to exclusively point to a portion of a binary, e.g., a section. It holds both the virtual address as well as the file offset.
