SectionHeaderIdx Type

Namespace: B2R2.FrontEnd.BinFile.ELF
Assembly: B2R2.FrontEnd.BinFile.dll
Base Type: obj
All Interfaces: IEquatable<SectionHeaderIdx> , IStructuralEquatable , IComparable<SectionHeaderIdx> , IComparable , IStructuralComparable

Every symbol table entry is defined in relation to some section. This member holds the relevant section header table index.
