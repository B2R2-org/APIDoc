B2R2

BinReader Type

Namespace: B2R2
Assembly: B2R2.Core.dll
Base Type: obj

BinReader abstracts away the process of reading byte sequences. This is the base class for BinReader, which has two kinds of implementations: little-endian and big-endian BinReader.
